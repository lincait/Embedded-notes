# 顺序表

## 形式

用一组地址连续的存储单元，依次存放线性表中的数据元素。存放的数据按顺序一个挨着一个。

只要知道下标，就能立即访问对应元素。但在指定位置操作时，需要大片移动后续元素。

## 定义

```c
#define MAXSIZE 100
typedef struct {
    int data[MAXSIZE];
    int length;    // 当前长度
} SeqList;
```

这个结构体就是整个顺序表。可以直接利用当前长度`length`和元素`data`的一一对应来进行增删改查。

# 链表

链表在内存中不要求连续存储，增删高效。

**每个节点都有自己的数据域、指针域以及节点自身的地址。**只要通过给节点的指针域赋值，让各个节点连接起来，后续就可以通过移动一个指针来访问。

**因此头指针作为链表入口要固定不动，增删改查时创建另外的指针来操作，避免丢失链表。**除非要改变头节点本身（需要传二级指针）。

## 单向链表

### 定义

```c
typedef struct Node {
    int data;           // 数据域
    struct Node* next;  // 指针域，指向下一个节点
} Node;
```

### 有头和无头

有头：首地址`head`指向头节点，`head->next`才指向第一个有效节点。

无头：首地址`head`直接指向第一个有效节点。不需要额外初始化，只要定义一个指向空的指针就行

由上述区别导致两种链表在操作实现上有所不同，尤其在边界。注意当函数需要修改外部头指针**本身**的指向时，要在函数返回值把地址返回或用二级指针。

### 循环和不循环

不循环：尾节点的`next`指向空，只能按顺序向后遍历，要访问前面的只能从头再来。

循环：尾节点的`next`指向头节点，无头时指向第一个有效节点。

### 实现

以**有头不循环**为例：

```c
// 初始化，即创建一个头节点
Node *head = malloc(sizeof(Node));	// 后续判空省略
head->next = NULL;

cur = cur->next;		// 向后移动，遍历

// 在cur后面插入新节点newnode
newnode->next = cur->next;	
cur->next = newnode;

// 删除cur的后继节点，找到要删的前驱就要停下操作
Node *tmp = cur->next;	
cur->next = tmp->next;
free(tmp);
```

## 双向链表

### 定义

```c
typedef struct Node
{
    int data;
    struct Node *prev;	//指向前驱
    struct Node *next;	//指向后继
}Node;
```

为了通用性，可以把头节点单独定义，能存任意类型数据，改成：

```c
typedef struct Node
{
    struct Node *prev;	
    struct Node *next;	
    char data[1];		//占位符，让节点的数据紧跟在指针之后，需要注意内存对齐。C99用data[]更好。
}Node;

typedef struct
{
    int size;			//存放节点的数据域所占的空间，由用户决定
    struct Node head;	//头节点
}LIST;

// 这种写法申请时要加上size
newnode = malloc(sizeof(Node) + LIST->size);
```

在实际中，往往采用通用性更强的**`linux`内核链表**（带头双向循环）：

核心思想：把链和数据彻底解耦。链表只负责指针连接，不关心数据类型；数据只负责存值，不关心如何连接。

- 定义链表结构：

```c
struct list_head {
    struct list_head *prev;
    struct list_head *next;
};
```

- 使用方式：不在链表节点里放数据，而是**在宿主结构体内部嵌入一个 `list_head` 成员**

```c
struct student {
    int id;
    char name[20];
    struct list_head list;   // 这个成员就是用来挂到链表上的钩子
};
```

- 遍历链表时，拿到的指针永远是指向 `struct list_head` 的。所以为了从钩子的地址反推出整个宿主的起始地址，内核使用 `container_of` 宏（即`list_entry`函数）。它根据钩子在宿主内固定的偏移量以及自己的地址计算出宿主的首地址。

### 有头和无头

区别和使用上与单向链表相同，一般使用有头。

### 循环和不循环

不循环：尾节点的`next`指向空。

循环：尾节点的`next`指向头节点，头节点的`prev`指向尾节点。实际工程最常用**有头双向循环链表**。

### 实现

以**有头循环**为例：

```c
// 初始化
Node *head = malloc(sizeof(Node));	// 判空省略
head->next = head;
head->prev = head;

// 遍历,向前或向后
cur = cur->next;
cur = cur->prev;

// 在cur后面插入新节点newnode,如果是在cur前面插入类似的就只要改前两行
newnode->prev = cur;			
newnode->next = cur->next;	
newnode->prev->next = newnode;
newnode->next->prev = newnode;

// 删除节点cur
cur->prev->next = cur->next;
cur->next->prev = cur->prev;
free(cur);
```
