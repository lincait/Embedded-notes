# 相关概念

栈存储的数据**先进后出**，实际操作时只能操作栈顶那一个位置。刚入栈的数据会在栈顶，并把前面的数据往下压一个位置。

# 顺序存储栈

## 说明

定义的一个结构体就是整个栈，数据按顺序存储。随着数据入栈，栈顶索引不断变大。

## 定义

```c
#define MAXSIZE
typedef struct
{
    int data[MAXSIZE];
    int top;	// 栈顶,数组索引
}sqstack;
```

## 实现

```c
// 初始化栈
sqstack *st = malloc(sizeof(sqstack));	// 后续判空省略
st->top = -1;	// 表示栈（数组）中没有元素

// 入栈，要先判断是否栈满 top==MAXSIZE-1
st->data[++ st->top] = data;

// 出栈，要先判断是否栈空 top==-1
data = st->data[st->top --];

// 遍历就跟数组一样，从0到st->top

// 释放内存
free(st);	
```

# 链式存储栈

## 说明

以有头链表为例，入栈和出栈都只操作头节点后面一个节点，即第一个有效节点为栈顶。

## 实现

- 初始化栈：初始化链表；
- 入栈：头节点后面插入新节点；
- 出栈：删掉头节点后面的节点；
- 释放：`free`全部节点
