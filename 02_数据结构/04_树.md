# 树的基本概念

结构：以节点为基本元素，以根节点为起始向下扩展。

高度：以最底层为0开始计，每加1层高度加1。

深度：根节点为第1层，其孩子为第2层，依此类推，树中节点的最大层数为深度。

度：一个节点拥有的子树个数，例如一个节点有两个子节点，度为2。

叶子节点：没有子节点的节点。

孩子、父亲、兄弟、堂兄弟等与人类关系一样。

# 二叉树

## 基本概念

每个节点最多有两个子节点并严格区分左右次序（左右孩子）。若父节点编号为`n`，则左孩子编号为`2n`，右孩子编号为`2n+1`。

满二叉树：所有节点都有左右孩子。

完全二叉树：叶子节点只出现在最后两层，且最后一层叶子都是左孩子。

## 平衡二叉树

### 概念

如果不断把数据插入到同一边并删除另一边的，数会退化为链表，查找效率降低。为了提高效率需要对树进行平衡。平衡条件为：左子树高度-右子树高度的绝对值不超过1。实际中可能对不同的需求有不同的平衡条件以及旋转操作。

### 失衡种类

- LL型：根节点的左孩子的左孩子引起的树不平衡，比如根节点的左孩子加了一个左孩子。需要进行根节点右旋。
- RR型：根节点的右孩子的右孩子引起的树不平衡，比如根节点的右孩子加了一个右孩子。需要进行根节点左旋。
- LR型：根节点的左孩子的右孩子引起的树不平衡。需要先让左孩子左旋，再进行根节点右旋。
- RL型：根节点的右孩子的左孩子引起的树不平衡。需要先让右孩子右旋，再进行根节点左旋。

### 平衡过程

根据平衡条件找到不平衡的节点，根据上述对应种类进行判断，做对应的旋转。

右旋（以节点A为例，其左孩子为B）：把A及其右子树作为B的右子树；B成为新的根；B原来的右子树变为A的左子树。

左旋（以节点A为例，其右孩子为B）：把A及其左子树作为B的左子树；B成为新的根；B原来的左子树变为A的右子树。

# 树的遍历

## 深度优先搜索

一条路走到黑，访问到底再**返回到上一节点**访问另一条路，不断循环直到遍历所有。根据根节点被访问的时机，分为：

- 前序遍历：**根** -> 左子树 -> 右子树
- 中序遍历：左子树 -> **根** -> 右子树
- 后序遍历：左子树 -> 右子树 -> **根**

## 广度优先搜索

层序遍历，访问顺序：第 1 层（根） -> 第 2 层 -> ... -> 第 N 层（从左到右）。

# 树状存储

## 顺序存储

一般用于完全二叉树，依次把对应编号（从1开始）的节点存入数组，某个位置上没有节点也要空出位置，这样索引能和节点直接对应。若层数多且不是完全二叉树的话需要浪费大量空间。

## 链式存储

对于二叉树：每个节点（结构体）的数据域存节点，两个指针一个指向左孩子，一个指向右孩子

### 定义

```c
typedef struct TreeNode
{
    int data;	
    struct TreeNode *left;		// 指向左孩子
    struct TreeNode *right;		// 指向右孩子
}TreeNode;
```

### 实现

以无头链表为例，根据前序进行构建。

```c
// 创建新节点
TreeNode *createNode(int data)
{
    TreeNode *newnode = malloc(sizeof(TreeNode));	// 判空省略
	newNode->data = data;	newNode->left = NULL;	newNode->right = NULL;
    return newnode;
}

// 递归构建树,函数能够构建一颗完整的树
void buildTree(TreeNode **root)
{	
    int tmp;
    scanf("%d",&tmp);
    
    if(tmp == -1)	// 以-1表示节点为空,不创建节点(假设数据都大于0)
        *root = NULL;
    else
    {
        *root = createNode(tmp);	// 先构建根节点
        
        buildTree(&((*root)->left));	// 再构建左子树
        buildTree(&((*root)->right));	// 最后构建右子树
    } 
}

// 遍历输出，以前序为例
void Traverse(TreeNode *root)
{
    if(root == NULL) return ;
    printf("%d ",root->data);
    Traverse(root->left);
    Traverse(root->right);
}
```

### 调用

```c
TreeNode *root = NULL;	// 初始化即定义一个空指针
buildTree(&root);		// 构建完成后root仍指向根节点
Traverse(root);
```
