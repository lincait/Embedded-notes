# 树的基本概念

结构：以节点为基本元素，以根节点为起始向下扩展。

高度：以最底层为0开始计，每加1层高度加1。

深度：根节点为第0层，其孩子为第1层，依此类推，树中节点的最大层数为深度。

度：一个节点拥有的子树个数，例如一个节点有两个子节点，度为2。

叶子节点：没有子节点的节点。

孩子、父亲、兄弟、堂兄弟等与人类关系一样。

# 二叉树

## 基本概念

每个节点最多有两个子节点并严格区分左右次序（左右孩子）。若父节点编号为`n`，则左孩子编号为`2n`，右孩子编号为`2n+1`。

满二叉树：所有节点都有左右孩子。

完全二叉树：叶子节点只出现在最后两层，且最后一层节点必须连续地靠左排列。

## 二叉搜索树

### 概念

搜索树能让查找、插入、删除等操作更加高效。以下为核心特征：

- 左子树上所有节点的值 都小于 根节点的值。
- 右子树上所有节点的值 都大于 根节点的值。
- 左右子树本身也分别满足上述性质。

### 主要操作

- 遍历：一般采用中序遍历，能直接得到有序序列

- 插入：新插入的节点一定是一个新的叶子。

- 删除：分以下几种情况

  1.待删除的节点p为叶子结点，可以直接删除；

  2.待删除的节点p只有左子树或只有右子树，将左子树的根或者右子树的根取代要删除节点位置；

  3.待删除节点的左、右子树均非空，首先找到节点p在中序序列的**直接前驱（或后继）节点s**，然后用**s的数据替代节点p的数据**，再将**节点s删除**。（这里找到待删节点的前驱后还要判断前驱是否有孩子，没孩子直接删，有孩子则把这个前驱的位置替换掉。这个前驱不可能会有右子树）

## 平衡二叉树

### 概念

如果不断把数据插入到同一边并删除另一边的，树会退化为链表，查找效率降低。为了提高效率需要对树进行平衡。平衡条件为：**树中任意节点的左右子树高度差绝对值都不超过 1**。实际中可能对不同的需求有不同的平衡条件以及旋转操作。

### 失衡种类

- LL型：根节点的左孩子的左孩子引起的树不平衡，比如根节点的左孩子加了一个左孩子。需要进行根节点右旋。
- RR型：根节点的右孩子的右孩子引起的树不平衡，比如根节点的右孩子加了一个右孩子。需要进行根节点左旋。
- LR型：根节点的左孩子的右孩子引起的树不平衡。需要先让左孩子左旋，再进行根节点右旋。
- RL型：根节点的右孩子的左孩子引起的树不平衡。需要先让右孩子右旋，再进行根节点左旋。

### 平衡过程

根据平衡条件找到不平衡的节点，根据上述四种对应种类进行判断，做对应的旋转，旋转后重新计算涉及操作部分的高度。

右旋（以节点A为例，其左孩子为B）：把A及其右子树作为B的右子树；B成为新的根；B原来的右子树变为A的左子树。

左旋（以节点A为例，其右孩子为B）：把A及其左子树作为B的左子树；B成为新的根；B原来的左子树变为A的右子树。

# 树的遍历

## 深度优先搜索

一条路走到黑，访问到底再**返回到上一节点**访问另一条路，不断循环直到遍历所有。根据根节点被访问的时机，分为：

- 前序遍历：**根** -> 左子树 -> 右子树。适合保存树
- 中序遍历：左子树 -> **根** -> 右子树。适合BST得到有序序列
- 后序遍历：左子树 -> 右子树 -> **根**。适合删除树

## 广度优先搜索

层序遍历，借助队列实现。访问顺序：第 1 层（根） -> 第 2 层 -> ... -> 第 N 层（从左到右）。

# 树状存储

## 顺序存储

一般用于完全二叉树，依次把对应编号（从1开始）的节点存入数组，某个位置上没有节点也要空出位置，这样索引能和节点直接对应。若层数多且不是完全二叉树的话需要浪费大量空间。

## 链式存储

对于二叉树：每个节点（结构体）的数据域存节点，两个指针一个指向左孩子，一个指向右孩子

### 定义

```c
typedef struct TreeNode
{
    int data;	
    struct TreeNode *left;		// 指向左孩子
    struct TreeNode *right;		// 指向右孩子
}TreeNode;
```

### 实现

- 以无头链表为例，创建新节点


```c
TreeNode *createNode(int data)
{
    TreeNode *newnode = malloc(sizeof(TreeNode));	// 判空省略
	newnode->data = data;	newnode->left = NULL;	newnode->right = NULL;
    return newnode;
}
```

- 递归构建树，该递归构建函数要求输入数据必须采用前序构建，且遇到 `-1` 表示该位置为空子树。

```c
void buildTree(TreeNode **root)
{	
    int tmp;
    scanf("%d",&tmp);
    
    if(tmp == -1)	// 以-1表示节点为空,不创建节点(假设数据都大于0)
        *root = NULL;
    else
    {
        *root = createNode(tmp);	// 先构建根节点
        
        buildTree(&((*root)->left));	// 再构建左子树
        buildTree(&((*root)->right));	// 最后构建右子树
    } 
}
```

- 遍历：

  层序遍历：使用队列，把根节点入队，再出队打印，并且每次有节点出队时，就把对应的左右孩子入队
  以下为前序遍历，中序后序只用调换顺序

```c
void Traverse(TreeNode *root)
{
    if(root == NULL) return ;
    printf("%d ",root->data);
    Traverse(root->left);
    Traverse(root->right);
}
```

- 插入和删除节点：需要根据具体树的类型来操作。

### 调用

```c
TreeNode *root = NULL;	// 初始化即定义一个空指针
buildTree(&root);		// 构建完成后root仍指向根节点
Traverse(root);
```
