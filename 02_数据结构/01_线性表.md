# 顺序表

## 形式

用一组地址连续的存储单元，依次存放线性表中的数据元素。存放的数据按顺序一个挨着一个。

## 优缺点

只要知道下标，就能立即访问对应元素。

但在指定位置操作时，需要大片移动后续元素。

## 实现

```c
#define MAXSIZE 100
typedef struct {
    int data[MAXSIZE];
    int length;    // 当前长度
} SeqList;
```



利用当前长度和元素的一一对应来进行增删改查。

# 单向链表

## 说明

链表在内存中不要求连续存储，增删高效。

**每个节点都有自己的数据域、指针域以及节点自身的地址。**只要通过给节点的指针域赋值，让各个节点连接起来，后续就可以通过移动一个头指针来访问。但要访问前面的只能从头再来。

**因此头指针作为链表入口要固定不动，增删改查时创建另外的指针来操作，避免丢失链表。**除非要改变头节点本身。

## 实现

### 定义

```c
typedef struct Node {
    int data;           // 数据域
    struct Node* next;  // 指针域，指向下一个节点
} Node;
```

### 基本操作

```c
void func(Node *cur)		
{
    Node *newnode = malloc(sizeof(Node));	//创建新节点
    
    cur = cur->next;		//向后移动，遍历
    
    //在cur后面插入新节点newnode
    newnode->next = cur->next;	
    cur->next = newnode;
    
    //删除cur的后继节点，找到要删的前驱就要停下操作
    Node *tmp = cur->next;	
    cur->next = tmp->next;
    free(tmp);
}
```

## 有头和无头

有头：首地址`head`指向头节点，`head->next`才指向第一个有效节点。

无头：首地址`head`直接指向第一个有效节点。

由上述区别导致两种链表在操作实现上有所不同，尤其在边界。注意无头的把`head`传入函数时，不能直接操作`head，`要在函数返回值把地址返回或用二级指针。

## 单向循环链表

尾节点的`next`指向头节点，无头时指向第一个有效节点。

尾节点判断：`node->next == head`。

创建单向循环链表：先建一个节点，让它的`next`指向自己，后续的节点用插入操作。

# 双向链表

## 说明

每个节点包含数据、指向前一个节点的指针、指向下一个节点的指针。

## 实现

### 定义

```c
typedef struct Node
{
    int data;
    struct Node *prev;	//指向前驱
    struct Node *next;	//指向后继
}Node;
```

为了通用性会把头节点单独定义，需要额外为数据申请一次内存，为数据域赋值时用`memcpy`：

```c
typedef struct Node
{
    void *data;			//类型由用户决定
    struct Node *prev;	
    struct Node *next;	
}Node;

typedef struct
{
    int size;			//存放节点的数据域所占的空间，方便为其申请内存。由用户决定
    struct Node head;	//头节点
}LIST;
```

**在上述基础上，可以把节点定义改成：**

```c
typedef struct Node
{
    struct Node *prev;	
    struct Node *next;	
    char data[1];		//占位符，让节点的数据紧跟在指针之后，需要注意内存对齐。C99用data[]更好。
}Node;

//这种写法只需申请一次内存，申请时要加上size
newnode = malloc(sizeof(Node) + LIST->size);
```

### 基本操作

```c
//以下为双向循环链表操作，非循环时注意判断前驱或后继为空的情况
void func(Node *cur)		
{
    //遍历,向前或向后
    cur = cur->next;
    cur = cur->prev;
    
	//在cur后面插入新节点newnode,如果是在cur前面插入类似的就只要改前两行
    newnode->prev = cur;			
    newnode->next = cur->next;	
    newnode->prev->next = newnode;
    newnode->next->prev = newnode;
    
    //删除节点cur
    cur->prev->next = cur->next;
    cur->next->prev = cur->prev;
    free(cur)；

}
```

## 双向循环链表

头节点的前驱指向尾节点，尾节点的后继指向头节点。

实际工程最常用**有头双向循环链表**。工程常用判空：`head->next == head && head->prev == head`
