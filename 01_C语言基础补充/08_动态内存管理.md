# malloc、ralloc和realloc 

## 堆和栈

堆：程序员**手动申请和释放**，存储动态分配的对象、数据结构。

栈：编译器**自动分配和释放**，存储局部变量、函数参数、返回地址。

**原则：谁申请谁释放**

## malloc 

### 基本使用

```c
int *p = NULL;
p = malloc(sizeof(int));
......
free(p);
```

申请某一大小的空间（字节数），返回该空间的起始地址，失败返回`NULL`。

分配的内容未初始化。

### 实现数组

```c
int *p;
int num = 5;

p = malloc(sizof(int) * num);
```

此时`p`就相当于有5个元素的数组的首地址，可以当作可变长数组用。
