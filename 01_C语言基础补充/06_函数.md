# 函数的定义

## 形式

数据类型	函数名（数据类型 形参名，......）

返回值的数据类型可以看成是函数的类型

**函数名会退化为函数对应的地址**，但对其解引用还是函数本身而不是返回值。

## `main`函数

`main`函数有时定义为：`int main(int argc,char *argv[])`，其中`argc`表示命令行运行时传入的参数的**个数**（包括程序名本身），`argv`是一个字符型指针数组，每个元素都指向一个字符串（命令行的每个参数）。

例如：`./main hello world 123`，则`argc`的值为4，`argv[0]`为`./main`，`argv[4]`为NULL，作为结束标志。

# 函数的传参

## 值传递

把实际参数的**值**传入给函数的形式参数，在函数中对形参的各种操作都不会影响实参。

## 地址传递

把实际参数的**地址**传给函数的形式参数，函数的形参**用指针来接收**，因此该指针就指向了实参，所以在函数中通过访问该指针进行的操作可以影响到实参。

# 函数的调用

## 嵌套

## 递归

一个函数在内部直接或间接的调用自身，直到满足终止条件。一层一层进，以栈的形式一层一层`return`返回。

### 基本要素

1. 边界条件：什么时候直接返回？
2. **递归假设：假设函数对更小规模的问题已经能够正确工作（关键）**
3. 归约步骤：如何利用“小规模问题的解（n-1）”构造出“当前规模的解（n）”？

必要时可画出调用顺序以及参数的“树状图”来解读代码。

# 函数与数组

**分清楚传入的实参是什么类型，是值、地址、行指针还是指向地址的指针，针对它们分别用值、指针、数组指针和二级指针来接收。**

例如：`int a		int *a 		int a[][N]	int *a[]`，都传入`a`

则形参：`int a		int *p		int (*p)[N]	int **p/int *p[]`

## 函数与一维数组

一维数组传参时，形参可以有两种方式接收：`int *p`或`int p[]`（`[]`相当于`*`），但是两种都一样是用一个指针接收。在函数中对`p`的操作可以影响到实参。

## 函数与二维数组

二维数组`a[M][N]`传参时，如果传入的是`a`，则传入的是一个行指针，形参需要用**数组指针**来接收`int (*p)[N]`或`int p[][N]`。

如果传入的是`&a[0][0]`、`a[0]`或`*a`时，则传入的是列指针，可以直接用`int *p`接收，但区分不了行列。

## 函数与字符数组

传入的实参为`char str[]`的数组名`str`，形参用`char *p`接收。

# 函数与指针

## 指针函数

指针类型的函数，由于函数返回值相当于类型，因此返回值为指针，如`int *func(int a)`。

## 函数指针

指向函数的指针，定义为：函数类型  （*指针名） （形参）

如`int (*p)(int)`，形参名可以省略。赋值时：`p = func`，函数名即为函数的地址。使用时，`p`可以代替函数名。

函数的某个参数是一个函数时，实参传入的是函数名（地址），则形参就可以用函数指针来接收。

可以在结构体里面放函数指针，实现类似面向对象中类的方法。

## 函数指针数组

函数指针数组是一个数组，里面每个元素都是指向函数的指针。和指针数组的区别就是它元素不只是指针，而是指向函数的指针。

定义为：函数类型  （*数组名【下标】） （形参）

如`int (*arr[N])(int)`
