# 数据类型

## 存储形式

大小端存储：大端是低位存在高地址，小端是低位存在低地址（x86、ARM处理器采用）。

以二进制进行存储，即按位存储，但C语言不能直接赋值二进制。

整型：以补码形式存储

浮点型：形式为3.14 = 0.314*10^1。内存中最高位为符号位，中间为指数部分，低位存储小数部分。

字符型：以ASC码的二进制存储

## 类型转换

隐式：不同类型混合运算后，结果自动靠拢精度高的。

显式：强制

## 特殊性

`float` 型并不是一个精确的表示，不能用 `==` 来判断，而是约等于。

`char` 型是否有符号是未定义的。

# 常量和变量

## 常量

### 字符常量：

由单引号引起来的单个字符，如 `‘a’，‘\n’，‘\0’，‘\015’`(八进制)，`‘\x7f’`(十六进制)。

### 字符串常量：

由双引号引起来的，如``“”，“a”，“asdf\n\012”``。

### 标识常量：

`#define`，在预处理阶段，只是单纯把宏名替换为宏体，不会做语法检查。

使用涉及到传参和运算时，尽量带上括号。

在为了节约资源的情况下一般使用带参宏而不是函数，如内核。

## 变量

[存储类型]	数据类型	标识符	=	值

### 数据类型：

基本数据类型、构造类型

### 存储类型：

`auto`：默认，自动分配和回收空间。

`register`：建议型关键字，需要满足各种要求，最终是否分配到寄存器由编译器决定。

`static`：自动初始化为0或空值，其值有继承性。用于修饰函数和全局变量时无法跨文件访问

`extern`：说明型，不会改变变量类型。在另外的文件声明要使用的变量（扩展作用域）

# 运算符和表达式

## 运算符

运算符在前，先进行计算，在取变量值使用。变量在前，先取变量值使用，在进行计算运算符。

三目运算：`op1 ? op2 : op3`，如果op1为真则取op2，否则取op3。

### 位运算

`<<` : 左移右补0，相当于乘2	`>>` : 右移左补0，相当于除2

`~` : 按位取反	`|` : 按位或	`^` : 按位异或	`&` : 按位与

和1异或相当于取反，和0异或相当于不变

将操作数第n位置1，其他位不变：`num = num | 1 << n`;

将操作数第n位清0，其他位不变：``num = num &  ~(1 << n)``;
