# 指针和指针变量

## 指针变量

指针变量是能够保存地址的变量，放在其中的内容就是指针（具有指向作用），指针变量的类型必须和它指向的值的类型相同。

## 指针定义以及说明

`int *p = &i`，其中指针变量`p`存放了`i`的地址，**指向了`i`**。即`p`等于`i`的地址，`*p`等于`i`，`&p`等于`p`的地址。 

定义时p是指针名，`int *`是类型。

单个指针在某一平台下所占用的字节数是确定的，和几级，类型无关。

# 空指针、野指针以及空类型指针

空指针：定义指针时还没想好要指向哪，可以定义为`int *p = NULL`。

野指针：当前指针的指向不确定，或没有指向。

空类型：`void *p = NULL`，不确定指针指向什么类型时使用，并且能跟其他类型互相赋值

# 指针与数组

## 指针与一维数组

`int a[N]; int *p = a;`

此时`p`即为`a[N]`的首地址，指向`a[0]`，`p+i`为`a[i]`的地址，指向`a[i]`。

因此：`a[i] == *(a+i) == *(p+i) == p[i]`	`&a[i] == a+i == p+i == &p[i]`

a和p的区别：**a是常量，p是变量**

## 指针与二维数组

`int a[M][N]`，**`a`为指向行的地址**，相当于是`a[M]`的首地址，`a+i`是把指向的位置按行移动；`int *p`，`p+i`只是把指向向后移动`i`个位置，相当于按列移动，因此不能直接赋值`p = a`。

因此：`&a[i][0] == a[i]`	`&a[0] == a`	

`a[i] == *(a+i)`	`a[i][j] == *(*(a+i)+j)`

## 指针与字符数组

### 1.字符指针初始化字符串

` char *str = "hello"`，此处初始化是把指针str指向了hello的首地址，可以修改str的指向：`str = "world"`。但不能直接修改`str[0] = 'w'`，所以不能用`strcpy`。

### 2.字符数组初始化字符串

`char str[] = "hello"`，此处初始化值被赋值给字符数组str，修改时可以用`strcpy`，但`str`是存放地址的常量，不能用`str = "world"`修改。

### 区别

1是字符串被放在了固定的只读空间，所以可以用指针指向不同的空间；2是先为数组开辟了一块可写空间，在这个空间上存入字符串，因此可以更改这块空间上的存放内容。

# 指针数组和数组指针

## 指针数组

定义：`int *arr[3]`，这是一个数组，里面存放的每个元素都是类型为`int *`的指针。

## 数组指针

定义：`int (*p)[N]`，这是一个指针，指向一个数组，它在移动时是**按数组大小进行移动**。它相当于是一个行指针，而普通的p相当于列指针。

对于`int a[M][N], p = a`，`a+i`和`p+i`是相同的。

# Const与指针



## 指针常量

`int *const p`，指针的**指向不能变**，但是指向的量本身可以变

## 常量指针

`const int *p`，指针**指向的量本身是不能变**的，但是指向可以变

## 区分

按`*`和`const`出现的先后顺序确定是指针和常量的先后顺序

看`const`修饰什么，只修饰`p`说明不能变的是指针本身，修饰`*p`说明不能变的是指向的目标本身。

## 注意项

`const`只是限制了它修饰的对应量不能变，但可以通过其他变量间接改变

`const int *cont p`指向和指向的目标都不能变

# 多级指针

`int a = 10;`

`int *p = &a;       // 一级指针初始化`

`int **q = &p;     // 二级指针初始化`

二级指针是指向地址的指针，`*q = p; **q = a`，q中存放p的地址，p中存放a的地址
