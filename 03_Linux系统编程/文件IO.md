# C标准IO库函数

## 打开/关闭文件

- `FILE *fopen(char *, char *)`：

  参数1为要打开的文件；参数2为打开模式，分为只读、只写、只追加写等。其中，只读模式若文件不存在会返回空且不创建文件。

  返回值为结构体指针`FILE *`，即文件。

- `int fclose(FILE *)`：

  参数为需要关闭的文件，成功返回0，失败返回EOF（负数）。

## 向文件写入数据

- `int fputc(int, FILE *)`：

  参数1为写入的数据（一个字节）；参数2为写入的文件，写在哪取决于文件打开模式。

  成功返回写入的值，失败返回EOF。

- `int fputs(char *, FILE *)`：

  参数1为需要写入的字符串，参数2为写入文件。写入成功返回非负整数。

- `int fprintf(FILE *, char *, ...)`：

  参数1为写入文件；参数2为写入的字符串（可以带格式）；可变参数3为格式对应的变量。

  成功返回写入字符的个数。

## 从文件读取数据

- `int fgetc(FILE *)`：

  参数为要读取的文件。返回读到的一个字节，出错或到文件末尾返回EOF。

  需要定义一个整型数据来接收，要输出它时可以转换成字符型。

- `char *fgets(char *, int, FILE *)`：

  参数1用于存放读取的数据；参数2为要读取的长度；参数3为读取的文件

  成功返回字符串，失败返回空。

- `int fscanf(FILE *, char *, ...)`

  参数1为读取文件；参数2为格式占位符，用于接收读取的数据；可变参数3用于存放接收的数据

  返回成功匹配到的数据个数（为格式占位符匹配到的个数）

## 标准输入输出和错误

- `stdin`：`FILE *`类型的标准输入流指针，指向终端的输入。终端的输入会流向它
- `stdout`：`FILE *`类型的标准输出流指针，指向屏幕。它的数据会流向屏幕输出
- `stderr`：和`stdout`一样都会把数据流到屏幕输出

# 系统调用

系统调用是操作系统内核提供给应用程序的API，使其可以间接访问硬件资源。

常用系统调用（`ssize_t`为有符号整数，`size_t`为无符号整数）：

- `int open(char *, int, ...)`：

  参数1为打开文件的路径；参数2为打开模式，需要多个选项时用按位或；可变参数为创建文件的权限（八进制）

  成功返回文件描述符，失败返回-1

- `ssize_t read(int, void *, size_t)`：

  参数1为要读取的文件描述符；参数2用于存放读取的数据；参数3为要读取的最大字节数。

  成功返回读取的字节数，失败返回-1

- `ssize_t write(int, void *, size_t)`

- `int close(int)`

- 库函数`exit`和系统调用`_exit`

  两个都是退出进程，库函数的退出会进行清理操作，系统调用的退出不会执行任何清理。
